# Context Systems Group
The Context Systems Group is part of the AI Engineering organization focused on context gathering tools used to power our AI systems
## Team Overview[](https://handbook.gitlab.com/handbook/engineering/ai/context-systems/#team-overview)
The Context Systems Group is part of the AI Engineering organization focused on context gathering tools used to power our AI systems.
## Our Projects[](https://handbook.gitlab.com/handbook/engineering/ai/context-systems/#our-projects)
The Context Systems team owns and is actively working on the following projects:
  * [Knowledge Graph](https://handbook.gitlab.com/handbook/engineering/architecture/design-documents/gitlab_knowledge_graph/) - Semantic understanding and relationships within codebases.
  * [Codebase Semantic Indexing](https://handbook.gitlab.com/handbook/engineering/ai/ai-coding/codebase_semantic_indexing/) - Advanced code search and discovery capabilities using embeddings.
  * [Repository X-Ray](https://handbook.gitlab.com/handbook/engineering/ai/ai-coding/repository_xray/) - Repository analysis and metadata extraction for enhanced code suggestions context.
  * [Duo Context Exclusion](https://handbook.gitlab.com/handbook/engineering/ai/ai-coding/duo_context_exclusion/) - Filtering of sensitive or irrelevant code context.
